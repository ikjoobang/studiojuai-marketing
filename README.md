# 🚀 AI 마케팅 자동화 플랫폼 (Universal Marketing AI Platform)

## 📋 프로젝트 개요

**모든 업종에 적용 가능한 범용 AI 마케팅 자동화 생태계**

Studiojuai 마케팅 최적화 플랫폼은 30개의 전문 AI 봇이 업종별로 자동 적응하며,
실시간으로 매장 마케팅을 최적화하는 완전 자동화 마케팅 솔루션입니다.

## 🎯 핵심 기능

### 🔄 **동적 업종 선택 시스템**
- 카페 / 치킨 / 한식 / 미용실 / 피자 / 디저트 / 기타 업종 지원

## 🛠️ 기술 스택

### **프론트엔드**
- HTML5 / CSS3 / JavaScript (ES6+)
- Chart.js (데이터 시각화)
- Tailwind CSS (스타일링)
- Font Awesome (아이콘)

### **API 연동**
- **네이버 API**: 검색 API, 지도 API, 플레이스 API
  - 로컬 검색 API로 실제 매장 데이터 자동 수집
  - 매장명 + 위치 기반 자동 검색
  - 실시간 카테고리, 주소, 설명 정보 활용
- **GPT API**: OpenAI ChatGPT API
  - GPT-4 모델 기반 AI 봇 실행
  - 업종별 맞춤 프롬프트 생성
  - 네이버 플레이스 실제 데이터 통합 분석
- **RESTful API**: 자체 데이터 관리 시스템

### **데이터베이스**
- 내장 테이블 스키마 (매장, 봇 실행 기록, 분석 데이터)

### **🔒 보안 시스템**
- **API 키 암호화**: XOR + Base64 암호화 저장
- **Rate Limiting**: GPT(10/분), 네이버(25/분), 봇(30/5분)
- **입력값 검증**: XSS 방지, SQL Injection 차단
- **보안 로깅**: API 호출, 의심 활동 추적
- **CSRF 보호**: 세션 기반 토큰 검증

## 🔑 API 설정 및 연동 가이드

### **✅ 네이버 API 설정 (완료)**
```javascript
// js/data-manager.js에 이미 설정됨
const NAVER_CLIENT_ID = 'fUhHJ1HWyF6fFw_aBfkg';
const NAVER_CLIENT_SECRET = 'gA4jUFDYK0';
```

**사용 가능한 API:**
- ✅ 네이버 검색 API (지역 검색)
- ✅ 네이버 로컬 검색 API (매장 정보 자동 수집)
- ✅ 네이버 지도 API
- ✅ API 키 설정 완료
- ⚠️ CORS 문제로 서버 프록시 필요 (현재 시뮬레이션 모드)

**네이버 플레이스 연동 방식:**
1. **매장 정보 입력 시** 네이버 플레이스 URL 필드 입력 (선택 사항)
2. **봇 실행 시** 자동으로 네이버 로컬 검색 API 호출
3. **검색 방식**: `매장명 + 위치` 조합으로 자동 검색
4. **수집 데이터**: 카테고리, 주소, 설명, 좌표, 전화번호
5. **프롬프트 통합**: AI 봇 분석 시 실제 매장 데이터 자동 반영

### **✅ GPT API 설정 (UI 구현 완료)**
1. **OpenAI API 키 발급**: https://platform.openai.com/api-keys
2. **설정 UI에서 입력** (권장):
   - 메인 화면 우측 상단 ⚙️ 설정 버튼 클릭
   - GPT API 키 입력 (`sk-proj-...` 형식)
   - "저장" 버튼 클릭
   - "테스트" 버튼으로 작동 확인
3. **또는 브라우저 콘솔에서 설정**:
   ```javascript
   securityManager.setApiKey('gpt', 'sk-your-api-key-here');
   ```

**GPT API 사용 예시:**

### **⚠️ 주의사항**
1. **API 연동**: 실제 GPT API 키 설정 필요 (설정 페이지에서 입력)
2. **네이버 API**: CORS 문제로 서버 프록시 필요 (현재 시뮬레이션 모드)
